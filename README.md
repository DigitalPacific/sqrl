# sqrl

```
Squirrel is a collection of CLI tools which help simplify the work required for CI tasks.

Usage:
  squirrel [command]

Available Commands:
  file        Get and set operations for json, yaml & txt files
  help        Help about any command

Flags:
  -h, --help   help for squirrel

Use "squirrel [command] --help" for more information about a command.
```
